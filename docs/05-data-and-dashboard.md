# Data first, then dashboard queries

Outcome
Define the canonical queries the dashboard will use before building the UI.

Queries MVP
- total pageviews last 24h, 7d, 30d
- unique visitors estimate last 24h, 7d, 30d
- sessions estimate
- pageviews timeseries
- top pages by pageviews
- top referrers
- geo distribution by country, region, city
- filter by project_id

UI MVP apps/dashboard
- project selector
- overview cards
- timeseries chart
- tables for top pages and referrers
- geo list or map ready dataset
